Webcam 구동하기

http://wiki.ros.org/usb_cam 

> sudo apt-get install ros-indigo-usb-cam ros-indigo-image-view

# Camera를 연결하고...
> roslaunch usb_cam usb_cam-test.launch
새로운 터미널에서.. 

> source (my workspace)/devel/setup.bash
> rostopic list

/rosout_agg

Image Connector Fork하기

https://github.com/jihoonl/image_connector_example

Image Connector clone 및 컴파일하기


> source (my workspace)/devel/setup.bash

> cd (my workspace)/src

> git clone (Your forked image connector repository)

> catkin build

실행 및 webcam 연동하기

터미널에서 

> source (my workspace)/devel/setup.bash
> roslaunch usb_cam usb_cam-test.launch
새 터미널에서 

> source (my workspace)/devel/setup.bash
> rosrun image_connector image_connector_node image:=(camera topic)
