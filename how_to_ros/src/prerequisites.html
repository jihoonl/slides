Prerequisites

Keywords

  • indigo : the latest ROS release version
  • catkin : the current ROS build system
  • catkin package

  • topic : pub/sub based message format for async. data exchange
  • service : request/response based message format for sync. data exchange
  • rosnode : a single ros program.(e.g camera driver, teleop commander)
  • rosrun : runan executable in an arbitrary ROS package without having to cd(or roscd).
  • roslaunch : runs multiple ROS nodes and set parameters on the parameter server
