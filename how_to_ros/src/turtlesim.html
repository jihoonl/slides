Catkin Python 패키지 만들기1

CMakeLists.txt 
cmake_minimum_required(VERSION 2.8.3)
project(twist_commander)

find_package(catkin REQUIRED COMPONENTS
  geometry_msgs
  rospy
)
catkin_python_setup()
catkin_package()
setup.py 
#!/usr/bin/env python

from distutils.core import setup
from catkin_pkg.python_setup import generate_distutils_setup

d = generate_distutils_setup(
    packages=['twist_commander'],
    package_dir={'': 'src'},
)

setup(**d)

컴파일 및 실행하기

Terminal에서 
> source (my workspace)/devel/setup.bash
> roscore
새 Terminal에서 
> source (my workspace)/devel/setup.bash
> rosrun turtlesim turtlesim_node
새로운 터미널에서 
> source (my workspace)/devel/setup.bash
> rostopic echo /turtle1/cmd_vel
또다른 새로운 터미널에서 
# Move to (my workspace)
> source (my workspace)/devel/setup.bash
> catkin build
...
> rosrun twist_commander command_turtle.py
